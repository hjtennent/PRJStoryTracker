# PRJStoryTracker
An app for users to track news stories they're interested in over a long period of time.

`git clone 'https://github.com/hjtennent/PRJStoryTracker.git'`

`cd PRJStoryTracker/StoryTracker`

`npm i`

To run the application, Android Studio must be installed and an Android emulator should be setup. This emulator should be running API 29 and target Android 10.0. To run the app on the emulator, open the AVD Manager from the main Android Studio menu and press the green play button next to the suitable emulator. Next, run the following command:

`npx react-native run-android`

To run tests:

`npm test`
